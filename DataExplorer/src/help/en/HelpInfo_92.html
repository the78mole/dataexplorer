File and Data Set Comment

The file comment window allows to comment things relevant for all the data sets of this file. As starting point the actual date will be displayed. Initial this is the creation date of the file. This date might be modified of course, if the real data are created or logged at another day. If an object name is in use it will be displayed as well.

The file comment window will be visible by selecting its tab. The data set comment will be visible, in a lower table for all data sets part of this file.
Hint : Some part of the data set comment can be displayed as graphics header in graphics tab. The displayed header part gets separated by semicolon, colon or line break from the file comment. A hyphen will be used to separate the header and actual data set name. Changes will be saved by selecting a different tab. It is possible to edit in both both comment sections.

The data set comment is visible in graphical view below the graphs, if activated, and can modified here directly. Data and time is displayed in default and can be modified if required. This time mark should be used as indicator at what time this data was created. This comment should be used to write some notes which are data set specific, like events why this is different to others.

Hint : All comments are stored in the data file and might be displayed again.
