History Graphics View

This display tab window presents the log files on a time line â€“ starting with the most recent log file for the selected device and object.

The analysis uses OSD files of your device and selected object. Preferably they are accessed via link files in the object directory.

However, if you did not define an object or if you selected 'device oriented', the device folder is browsed. This is the reason why it is recommended arranging the log files in object folders which are sub-directories in the standard data path (refer to Preferences dialog 'Maintenance'). Having added an object (see last paragraphs here) a folder with the object name is created. The system will place a link file in such an object folder for future OSD files.

The history graphics view fetches its data whenever you click on the tab. This ensures that the graphics is refreshed whenever you add new log files to the directories.

Hint: Make this tab visible by activating the history in the Preferences dialog 'History'.

Depending on the selected device and and history settings deviations from the shown picture might be possible. The device configuration XML and the actual implementation state will have its influence as well.
The enhanced curve selector enables various statistical values in dependency of the history configuration.
It makes sense to store graphics templates for the history analysis view as well, since they hold different values compared to the standard graphics templates.

Additional capabilities of the graphics area context menu:

  • load a data set into the graphics view
  • delete a file (such as log files caused by tests)
  • set a file or data set to hidden (e.g. extreme values or outliers)
  • toggle hiding ('Selective history')
  • interactive analyses via 'Curve survey' in combination with 'Measurement' (see some way down here)

Hint: If a device supports a raw file import (see item 'Import' here) it might be possible to include these files into the analysis. This allows you to bypass creating OSD files.
