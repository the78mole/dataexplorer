iCharger206B, iCharger208B, iCharger306B, iCharger3010B - battery charger

There is device dialog for the iChargers. It is only required to establish the serial connection and it will start the gathering live data or start reading data from the device.

Hint: The used Baud rate of 128000 Baud can not be opened if a UNIX derived operating system is used, e.g. Linux or MAC OS X. This data rate doesn't have an even divisor. Linux user will have the opportunity to modify the kernel module source. Hints how to do that can be found in the Internet.

The iCharger 308DUO support enables to import device internal saved data. Use the import button. An file selection dialog will be opened. The iCharger must be connected as an USB mass storage device.
